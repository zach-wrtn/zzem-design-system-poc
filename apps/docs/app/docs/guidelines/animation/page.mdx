# Animation

ZZEM Design System의 애니메이션 가이드라인입니다.

## Principles

1. **목적이 있는 모션** — 사용자의 관심을 유도하거나 상태 변화를 전달할 때만 사용
2. **자연스러운 타이밍** — 150ms~300ms 범위, easing curve 적용
3. **접근성 고려** — `prefers-reduced-motion` 존중

## Duration Scale

| Token | Value | Usage |
|-------|-------|-------|
| `duration.instant` | 100ms | Micro-interactions (press feedback) |
| `duration.fast` | 150ms | Tooltips, small transitions |
| `duration.normal` | 250ms | Standard transitions |
| `duration.slow` | 350ms | Complex animations, page transitions |
| `duration.slower` | 500ms | Large overlays, bottom sheets |

## Built-in Animations

- **Button press**: opacity 변화 (instant)
- **Toast**: fade in/out (normal)
- **Dialog**: fade overlay (normal)
- **BottomSheet**: slide up (slow)
- **Switch**: thumb translate (fast)

## Reduced Motion

```tsx
import { AccessibilityInfo } from 'react-native';

const isReduceMotionEnabled = await AccessibilityInfo.isReduceMotionEnabled();
```

사용자가 모션 감소를 설정한 경우, 애니메이션을 즉시 전환으로 대체합니다.
